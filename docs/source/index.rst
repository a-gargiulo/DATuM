Welcome to DATuM's Documentation!
=================================

The present html pages document the `PUBLIC` API of DATuM, a code suite enabling the direct analysis of selected turbulence models, using the BeVERLI Hill [insert reference] stereoscopic PIV data.

This code was written and is currently maintained by Aldo Gargiulo. Aldo Gargiulo is currently a postdoctoral research associate at the Reacting Flow Laboratory of the University of Virginia. Aldo Gargiulo wrote DATuM in the context of his Ph.D. dissertation at Virginia Tech under the supervision of Dr. Todd K. Lowe and Dr. Christopher J. Roy.
